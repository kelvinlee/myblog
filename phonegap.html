Phonegap 学习中.

简单介绍一下PhoneGap,就是用html/css/js,来制作移动app. 好了 介绍完了.

我的电脑环境, Mac lion 10.8.4 首先需要安装Nodejs.

$ brew install node

安装完node之后,就可以直接安装Phonegap了.

$ sudo npm install -g phonegap

这里还有一个问题,安装之后可能直接使用phonegap不好使,原因是是安装到的目录是不对的.所以你还的转换一下目录,问了一下大神说是用ln命令[哥的linux实在不敢恭维.] 这里需要注意如果你可以直接在Terminal 下使用phonegap,就不需要以下命令了.

$ sudo ln -s /usr/local/share/npm/bin/phonegap /usr/local/bin/phonegap

下面这个是另外一个cordova,作用貌似和phonegap一样,但是不知道不安装phonegap,直接是哟个cordova好不好使.

$ sudo npm install -g cordova

不多说,和上面原因一样.

$ sudo ln -s /usr/local/share/npm/bin/cordova /usr/local/bin/cordova

安装基本就完成了,然后创建demo

$ cordova create hello com.example.hello HelloWorld

demo就创建好了,其中的www文件夹就是html/css/js的文件了,你就可以随便更改了.

这里还需要注意个事情,因为cordova和phonegap默认安装完都会有一个模版,是在创建demo的时候自动从git上下载的,但是这个模版的路径又存在错误.所以如果demo创建失败,根据失败时提示的路径用finder打开,把3.0.0里面多出来的一级目录移出来,这样就不会报错了.

然后就是加入ios/android等模块,这里需要进入到对应的模块才行.

$ cd hello
$ cordova platform add ios
$ cordova platform add android

加入模块之后,进行生成.

$ cordova build

加入多少模块,build就会生成多少,如果只想单一生成某个,可以在后面加入ios什么的

$ cordova build ios

还有更详细的命令,可以查看这里.
